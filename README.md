# HPC on FPGA

Hamming Product Code on FPGA.

> Designed for 16 bit width.

> Use 7,4 or 8,4 Hamming Code



## Requirement

use docker image at [.devcontainer/Dockerfile](.devcontainer/Dockerfile)

---

## Build or Run

```bash
./build --help
```
